¿Qué son las Dimensiones?

Las dimensiones en Tailwind CSS se refieren al tamaño y espacio de los elementos en tu interfaz de usuario. Puedes controlar el ancho, alto, márgenes y rellenos de los elementos utilizando clases de utilidad predefinidas.

Ancho (Width)

Tailwind ofrece varias clases para establecer el ancho de un elemento. Aquí algunos ejemplos:

Ancho: 25%
Ancho: 50%
Ancho: 75%
Ancho: 100%
                
<div class="bg-blue-500 text-white p-4 w-1/4">Ancho: 25%</div>
<div class="bg-blue-500 text-white p-4 w-1/2">Ancho: 50%</div>
<div class="bg-blue-500 text-white p-4 w-3/4">Ancho: 75%</div>
<div class="bg-blue-500 text-white p-4 w-full">Ancho: 100%</div>

Alto (Height)

Al igual que el ancho, Tailwind también proporciona clases para controlar la altura de un elemento. Aquí tienes algunos ejemplos:

Alto: 3rem
Alto: 6rem
Alto: 8rem
Alto: 100%
                
<div class="bg-green-500 text-white p-4 h-12">Alto: 3rem</div>
<div class="bg-green-500 text-white p-4 h-24">Alto: 6rem</div>
<div class="bg-green-500 text-white p-4 h-32">Alto: 8rem</div>
<div class="bg-green-500 text-white p-4 h-full">Alto: 100%</div>

Espaciado (Margin & Padding)

Puedes controlar los márgenes y rellenos de los elementos usando las clases de Tailwind. A continuación se muestran ejemplos de cómo aplicar márgenes y rellenos:

Margen: 1rem
Relleno: 2rem
                
<div class="bg-yellow-500 text-white p-4 m-4">Margen: 1rem</div>
<div class="bg-yellow-500 text-white p-8">Relleno: 2rem</div>

Dimensiones de Pantalla Completa

Puedes hacer que un elemento ocupe toda la pantalla utilizando clases como `w-screen` y `h-screen`:

Este es un elemento de pantalla completa

                
<div class="bg-red-500 text-white w-screen h-screen flex items-center justify-center">
    <h3 class="text-2xl font-bold">Este es un elemento de pantalla completa</h3>
</div>
